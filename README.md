Welcome to my **Road to Data Engineering** repository!

The goal of this repository is to:

- Practice SQL, Python, and Data Pipelines

- Work with Cloud platforms
- Create real-world Data Engineering projects
- Prepare for Data Engineering interviews

This repository will continuously evolve as I progress in my learning journey.
